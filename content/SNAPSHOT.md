# ai-cost-audit - Snapshot

## Purpose

ai-cost-audit helps estimate AI product costs across multiple LLM providers and presents the result as a calculator plus cost report.

## Current State

- The repo is very lightweight and currently centered on a static web experience
- There is a live HTML page and a sample PDF report artifact
- Structured project tracking files were missing and are now being added for Meridian

## What Exists

- Public-facing landing/calculator page
- Basic README positioning statement
- Sample report output

## Main Risks

- Pricing assumptions and calculation logic are not yet documented in a tracked operational format
- The project structure is minimal, which makes future maintenance and confidence harder
- Without explicit update workflow, estimates can drift away from reality over time

## Recommended Next Steps

1. Document the pricing and calculation assumptions
2. Decide whether this remains a static tool or evolves into an app
3. Improve input/output clarity for user trust
4. Update this snapshot whenever the public estimator meaningfully changes
